# projectweek-02
# Styled Portfolio Website

## 1. Project Overview

This project is a responsive personal portfolio website created using HTML and CSS.

The portfolio includes:
- Header and navigation bar
- About section
- Skills section
- Contact form
- Profile image
- External links

### Objective
The main objective of this project is to practice CSS styling, responsive design, Flexbox layout, hover effects, and webpage structure.

---

# 2. Setup Instructions

## Step 1:
Download or clone the repository.

## Step 2:
Open the project folder in VS Code.

## Step 3:
Open index.html in browser.

No additional installation is required because the project uses only HTML and CSS.
